Perl Text Analysis

This is a suite of Perl modules and scripts I wrote for a course in Humanities Computing I took at the University of Georgia. These scripts provide tools to analyze texts, and the modules allow users to create scripts that analyze texts in other ways. At this stage, this represents an early effort that is incomplete, ideosyncratic, and buggy, and I recommend waiting for the next reincarnation of these tools before using them. I hope to make it worth the wait.
