PTA Installation

In the interests of enforcing the "don't use this" policy, I haven't made an installation file. On the other hand, if you have the rights, it's not that difficult of a process:
  1. Create a directory "PTA" in the Perl lib directory (generally, /usr/local/lib/perl):

    2.         cd /usr/local/lib/perl
        mkdir PTA
  2. Move all the *.pm files in the distribution into the new directory:
            cd Wherever you unzipped the distribution
                mv *.pm /usr/local/lib/perl/PTA
That should be it. You may want to put the directory you plan to keep the files in permanently in your path, but that's not necessary.
